TEST CASE-NITIN KUMAR

Test Case: Place a Cash on Delivery Order Successfully

    Test ID: TC005

    Test Summary: Verify that a user can successfully place a Cash on Delivery order.

    Preconditions:
        User is logged in.
        Shopping cart has items.

    Test Steps:
        Open the Amazon website.
        Log in with valid credentials.
        Add items to the shopping cart.
        Navigate to the checkout page.
        Select "Cash on Delivery" as the payment method.
        Confirm the order.

    Browser: Chrome, Firefox, Safari

    Test Data: Valid user credentials, valid product in the cart.

    Expected Data: Order confirmation message is displayed, and the order is visible in the user's order 		   history. The system should acknowledge that payment will be made on delivery.
